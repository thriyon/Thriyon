# Thriyon Nexus — Freelance Platform Roadmap

This document outlines the complete architectural mapping of the pages required to transform **Thriyon Nexus** from an elegant landing experience into a fully functional, sovereign operating system for elite freelancers, studios, and high-end clients.

To align with the Thriyon aesthetic (*cinematic, premium, dark-mode, minimal*), the terminology shifts away from generic Upwork/Fiverr concepts to a more curated, elite ecosystem.

---

## 1. Public & Discovery Layer (The Cinematic Front)

These pages are public-facing, highly optimized for SEO, and designed to inspire clients to start elite projects.

| Route | Page Name | Description | Key Features |
| :--- | :--- | :--- | :--- |
| `/` | **The Gateway (Home)** | The landing and cinematic index (Active & Refined). | WebGL Orb, staggered text animations, Manifesto, live network orbit, call to action. |
| `/talent` | **The Index** | The elite talent and studio registry. | Fuzzy search, category filtering (brand, 3D, code), availability badges, sleek grids. |
| `/talent/[username]` | **The Showcase** | Immersive, high-fidelity portfolio. | Large media displays (videos, renders), background, list of sovereign skills, "Initiate Project" CTA. |
| `/how-it-works` | **The Method** | Curated explanation of our methodology. | Smooth scrolling, typography-driven manifesto, escrow details, curation standards. |

---

## 2. Freelancer Space (The Sovereign OS)

A private workspace optimized for deep creative work. Minimal distractions, maximum efficiency.

| Route | Page Name | Description | Key Features |
| :--- | :--- | :--- | :--- |
| `/dashboard/freelancer` | **Sovereign Dashboard** | Central operational command center. | Active contract list, financial earnings charts, pending briefs, focus timers. |
| `/dashboard/freelancer/profile` | **Studio Settings** | The editor for the public Showcase. | Media gallery uploads, hourly/project rate controls, custom links, availability switcher. |
| `/dashboard/messages` | **The Conduit (Messages)** | Real-time quiet communications. | Direct messages, thread search, integrated file drawers, smart notifications. |

## 3. Client Space (The Initiator OS)

Designed for high-end founders, VC firms, and modern brands looking for top-tier execution.

| Route | Page Name | Description | Key Features |

Where actual work, communication, and money live in a single unified interface.

| Route | Page Name | Description | Key Features |
| :--- | :--- | :--- | :--- |
